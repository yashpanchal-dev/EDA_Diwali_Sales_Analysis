# 🪔 Diwali Sales Analysis (EDA)

This project analyzes **Diwali sales data** to identify customer purchasing behavior, high-performing product categories, and demographic trends using **Python and data analysis libraries**.

The goal of this analysis is to help businesses understand **which customers buy more during the festive season and which products generate the highest sales**.

---

## 📌 Project Overview

Diwali is one of the biggest shopping seasons in India. This project performs **exploratory data analysis (EDA)** on a Diwali sales dataset to uncover insights about:

- customer demographics
- product demand
- purchasing patterns
- regional sales distribution

The analysis helps businesses optimize **marketing strategies, inventory planning, and customer targeting**.

---

## 🛠 Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Key Analysis Performed

### ✅ 1. Data Cleaning
- Removed null values
- Dropped unnecessary columns
- Converted data types for accurate analysis

### ✅ 2. Gender-Based Sales Analysis
- Analyzed purchase behavior by gender
- Identified which gender contributes the most to total sales

### ✅ 3. Age Group Analysis
- Examined which **age groups spend the most during Diwali**
- Identified key customer segments

### ✅ 4. State-wise Sales Analysis
- Found states with the highest purchase activity
- Compared total orders and sales by state

### ✅ 5. Marital Status Analysis
- Evaluated purchasing behavior based on marital status
- Identified major contributing customer groups

### ✅ 6. Occupation Analysis
- Identified professions with the highest spending patterns

### ✅ 7. Product Category Analysis
- Determined the **most popular product categories**
- Analyzed which products generate the highest revenue

---

## 📈 Insights from the Analysis

Some key findings from the project include:

- **Married women aged 26–35 years** contribute significantly to overall sales.
- Customers from **Uttar Pradesh, Maharashtra, and Karnataka** show high purchasing activity.
- **IT, Healthcare, and Aviation sector professionals** are among the top buyers.
- **Food, Clothing, and Electronics** are among the most popular product categories during Diwali.

## ✅ Conclusion

This project demonstrates how **Python-based data analysis can uncover meaningful business insights from retail sales data**. By analyzing customer demographics, product demand, and purchasing trends, businesses can better understand their target audience during the Diwali season.

The analysis highlights that **married women aged 26–35 years are the most active buyers**, particularly from states like **Uttar Pradesh, Maharashtra, and Karnataka**. Additionally, customers working in **IT, Healthcare, and Aviation sectors** show higher purchasing activity.

These insights can help companies improve **marketing strategies, customer segmentation, and inventory planning** for future festive seasons.
